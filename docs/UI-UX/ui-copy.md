**URL Errors**

**Robots restriction**
This site does not permit automated scraping.
Operation blocked.

**Concurrency violation**
Another run is currently in progress.
Wait before launching another run.

**QuotaExceeded**
Free-tier quota reached for today.
Try again tomorrow or upgrade limits.

**Not allowed domain**
This resource is blocked due to private or internal network access.
