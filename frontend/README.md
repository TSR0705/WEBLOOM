# Webloom Frontend

React frontend application for the Webloom platform.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will run on `http://localhost:5173`

## Build

```bash
npm run build
```

## Features

- Dashboard with job listing and stats
- Job details page with metadata and actions
- History timeline view with charts
- Version detail view with HTML rendering
- Version comparison with diff visualization

## Tech Stack

- React 18
- React Router v6
- TanStack Query (React Query)
- Axios
- TailwindCSS
- Chart.js
- Lucide React Icons
